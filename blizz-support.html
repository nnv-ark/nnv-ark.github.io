BLIZZ — Support

A native macOS app for tracking billable time and turning it into clean reports and estimates — a modern rebuild of iBiz.

Getting started

  • Add a client (or import one from Contacts), then create a project under it.
  • Break the project into tasks, and log work as job events — type them in by hand or run the floating Quick Timer stopwatch.
  • Mark work billed as you go; the reports and unbilled totals key off that flag.
  • Open Reports for time, earnings, client and project summaries, and VSK/tax breakdowns — export any of them as a PDF sheet or CSV.
  • Generate estimates and statements as polished A4 PDFs, and share them via Mail, Messages or AirDrop.

Working with your other apps

  • Calendar & Reminders. Link a project to a calendar so events sync in as job events, add job events to Calendar, or push project to-dos to Reminders. Access is requested the first time and can be changed in System Settings → Privacy & Security.
  • File Cabinet. Connect a project to a work folder and BLIZZ tracks the files inside it, with live updates.
  • Self-hosted data folder. Point BLIZZ at a folder of your choice (for example inside a synced folder like Dropbox); your data is mirrored there as plain JSON and re-imported at launch, so you can keep it on your own storage.
  • Hand off to RUKK. Export a project's billable time for invoicing in RUKK.

Common questions

Where is my data stored? Locally on your Mac, in BLIZZ's own database — plus, if you enable it, mirrored as JSON to the data folder you choose. Nothing is sent to us.

How do I back up? Use File → Back Up All Data to write a full JSON backup, and Restore from Backup to bring it back.

Can I change the tax rate? Yes — set the default VSK/VAT percent and invoice numbering in Settings; company details there appear on your PDFs.
